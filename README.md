# SilentTalk

Inclusive communication platform for deaf, mute, and illiterate individuals.

## Quick start

**Frontend**
```bash
npm install
npm run dev
```
Runs at http://localhost:3000

**Backend**
```bash
cd backend
pip install flask flask-cors
python app.py
```
Runs at http://localhost:5000

## Video calling
1. Click "Enable camera"
2. Generate or enter a room ID
3. Share the room ID with the other person
4. Both click "Join room" — connected

> Same-browser tabs only. Cross-device requires a WebSocket signaling server.
